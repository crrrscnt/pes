# Система расчета потенциальной энергетической поверхности с помощью квантовых алгоритмов

Дипломная работа. Система предназначена для расчета потенциальной энергетической поверхности (ПЭП) с помощью VQE и автоматического подбора гиперпараметров расчета для VQE с исопльзованием LinUCB; она расчитана на исследователей вычислительной химии.
