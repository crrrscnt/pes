# Potential Energy Surface Calculation System Using Quantum Algorithms

Bachelor's degree diploma. The system is designed to compute potential energy surfaces (PES) using the Variational Quantum Eigensolver (VQE) and to automatically tune VQE hyperparameters using LinUCB; it is intended for researchers in computational chemistry.


# Система расчета потенциальной энергетической поверхности с помощью квантовых алгоритмов

Дипломная работа (бакалавра). Система предназначена для расчета потенциальной энергетической поверхности (ПЭП) с помощью VQE и автоматического подбора гиперпараметров расчета для VQE с исопльзованием LinUCB; она расчитана на исследователей вычислительной химии.
